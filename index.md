# F80 M3 整備計画の全体像 — オーナーが知るべきこと、プロが見ていること

> 11万km超のF80 M3を長く・安全に・気持ちよく乗り続けるための整備哲学と実践。
> オーナー視点の「何をやるか」と、プロ視点の「なぜそう判断するか」の両面で解説する。

---

## はじめに：M3の整備は「普通の車」と何が違うのか

M3は普通の3シリーズと**部品点数も専用設計の割合も全く違う**。整備の考え方も変わる。

| 観点 | 普通の3シリーズ | M3 F80 |
|------|---------------|--------|
| 設計思想 | 快適性・耐久性のバランス重視 | 性能限界を引き上げるために各部を専用設計 |
| 部品の負荷 | 余裕を持った設計マージン | 性能のためにマージンを削っている箇所がある |
| 劣化の影響 | 少し劣化しても「なんとなく乗れる」 | 劣化が性能と安全に直結する |
| 整備費用 | 相場が安定・選択肢が多い | 専用部品が多く高額。作業も専門知識が必要 |
| 整備の考え方 | 壊れたら直す（事後保全）でもある程度成立 | **壊れる前に手を打つ（予防保全）が基本** |

**プロの視点：**
> 「M3は"速い3シリーズ"ではなく"公道を走れるレーシングカー"に近い。
> レーシングカーの整備計画で考えないと、痛い目を見る」

## 第1章：整備計画の3つの柱

### 柱1：安全を守る整備（止まる・曲がるを維持する）

**何を守るか：** ブレーキ、タイヤ、足回り、灯火類

車を「走らせる」ことよりも「止める」「制御する」ことのほうが安全上は重要。
M3は大パワーFRなので、止まれない・曲がれないは即座に事故に繋がる。

| 部位 | プロが見るポイント | オーナーが気づけるサイン |
|------|------------------|----------------------|
| ブレーキパッド | 残量（mm単位で管理） | 警告灯、ブレーキ鳴き、制動距離の変化 |
| ブレーキローター | 摩耗量＋偏摩耗＋クラック | ペダルの振動（ジャダー） |
| ブレーキフルード | 沸点測定（水分含有量） | ペダルの奥まり、効きの変化（体感しにくい） |
| タイヤ | 溝の深さ＋偏摩耗パターン＋ヒビ | 見た目、雨の日のグリップ感、ロードノイズ変化 |
| 足回りブッシュ | ヒビ・変形・ガタの有無 | ハンドリングの曖昧さ、異音、直進安定性の低下 |
| ハブベアリング | 回転時のガタ・異音 | 走行中の「ゴー」という低い音 |

**プロの視点：**
> 「ブレーキパッドの残量が何mmかは、プロなら持ち上げて10秒でわかる。
> でもオーナーは普段乗っていて少しずつ効きが落ちるから慣れてしまう。
> 警告灯が点く前に定期的に測るのが正解。点いてからでは"もう限界"の意味」

> 「足回りのブッシュは"壊れる"んじゃなくて"じわじわ死ぬ"。
> 毎日乗ってると劣化に気づかない。新品に替えたとき初めて
> "ああ、こんなに違うのか"と思う。11万kmの個体は例外なくヘタっている」


### 柱2：壊れると致命的なものを予防する整備

**何を守るか：** エンジン内部、冷却系、駆動系

M3特有の「壊れたら修理代が車両価値を超える」部品がある。これらは**壊れてから直す**のでは遅い。

```
【致命度マップ】

  修理代
  100万〜│  ×エンジンブロー   ×メカトロ故障
        │  （ロッドベアリング）
   50万〜│      ×クランクハブ
        │       スリップ後
   20万〜│  ×ウォーターポンプ
        │   故障→オーバーヒート
   10万〜│
        │
        ┼─────┬─────┬─────┬──→ 予防コスト
            1万   5万  10万  15万

  × = 壊れた場合の被害
  予防コストは壊れる前に交換した場合のコスト

  → 予防コスト ≪ 壊れた場合の被害。だから予防する
```

#### S55エンジンの「三大持病」とプロの対応方針

**1. クランクハブ（スピンドルベアリング）問題**

```
  クランクシャフト先端の構造：

  ──[クランクシャフト]──●──[ハブ]──[タイミングギア]
                        ↑
                    ここが滑る（スリップ）

  ハブがスリップすると：
  → バルブタイミングがズレる
  → 最悪、ピストンとバルブが接触 → エンジンブロー
```

- **プロの対応**: 予防的に強化キットを装着。「症状が出てからでは遅い」が常識
- **費用対効果**: 強化キット6〜9万円 vs エンジン修理50〜100万円以上
- **タイミング**: ウォーターポンプ交換と同時施工で工賃を共有できる

**プロの視点：**
> 「クランクハブは"壊れるかどうか"ではなく"いつ壊れるか"の問題。
> 11万km走っていてまだ対策してないなら、最優先で入れたほうがいい。
> ポンプ交換のときに一緒にやれば、工賃は追加1万円程度で済む」

**2. ロッドベアリング摩耗**

```
  クランクシャフト
  ═══════●═══════●═══════  ← 回転する
         ┃       ┃
     [ベアリング] [ベアリング]  ← 薄い金属の軸受け
         ┃       ┃
     コンロッド  コンロッド
         ┃       ┃
     ピストン   ピストン

  ベアリングが摩耗すると：
  → 金属同士が接触 → 異音 → 焼付き → エンジンブロー
```

- **プロの対応**: オイル交換のたびにオイルサンプルを採取し、成分分析に出す
- **見るべき数値**: 銅（Cu）、鉛（Pb）、アルミ（Al）の含有量
- **判断基準**: 数値が上昇傾向なら予防交換を検討（エンジン降ろし作業：30〜50万円）
- **対策の限界**: 「壊れる前に100%見つけられる」わけではないが、最善の監視手段

**プロの視点：**
> 「オイル分析は3,000〜5,000円。エンジンブローは100万円。
> 保険としてはこれ以上コスパの良いものはない。
> S55に限らず、高性能エンジンを高走行で乗るならオイル分析は必須の習慣」

**3. 電動ウォーターポンプの突然死**

```
  正常時：
  エンジン →(熱)→ クーラント →(ポンプで循環)→ ラジエーター →(冷却)→ エンジン

  ポンプ故障時：
  エンジン →(熱)→ クーラント →(循環停止)→ 温度急上昇 → オーバーヒート
                                              ↓
                                     ヘッドガスケット損傷
                                     ヘッド歪み
                                     最悪エンジン交換
```

- **プロの対応**: 8〜12万kmで予防交換。「動いているうちに替える」
- **同時交換推奨**: サーモスタット、エクスパンションタンク（プラ製で割れる）
- **費用**: ポンプ＋サーモ＋タンク＋クーラント＋工賃で8〜12万円程度

**プロの視点：**
> 「電動ウォーターポンプは前兆なしで死ぬ。昨日まで元気で今日突然ダメ。
> 高速道路でオーバーヒートしたら路肩に止める以外何もできない。
> 11万kmで未交換なら"よく持っている"レベル。早めに替えたほうがいい」


### 柱3：気持ちよく乗るための整備（性能を維持する）

**何を守るか：** エンジンレスポンス、シフトフィール、ハンドリング、乗り心地

壊れてはいないが、新車時の性能から確実に落ちている部分を回復させる整備。

| 部位 | 新車時 | 11万km（未整備の場合） | 整備後 |
|------|--------|----------------------|--------|
| エンジンレスポンス | シャープ | もっさり感 | ほぼ回復 |
| シフトフィール | カチッと入る | 渋さ・ショック | かなり改善 |
| ステアリングの正確さ | ダイレクト | 曖昧、遊びが大きい | 新車同等に |
| 乗り心地 | 硬いが不快ではない | ゴツゴツ or フワフワ | しなやかに |
| コーナリング | 一体感 | 不安感、ブレる | 一体感回復 |

```
【性能劣化のイメージ】

  性能
  100%│━━━━━╲
      │       ╲━━━━━━━╲
   80%│               ╲━━━━━━  ← 劣化は加速する
      │                         （複数箇所が同時に劣化）
   60%│
      │    ↑             ↑
      │  新車          11万km
      ┼────────────────────→ 走行距離

  一気にリフレッシュすると：

  性能
  100%│━━━━━╲           ╱━━━━
      │       ╲━━━━━━━╲╱  ← 足回り一式交換
   80%│
      │
   60%│
      ┼────────────────────→ 走行距離
```

## 第2章：プロの整備計画の立て方

### 2-1. 「系統別」に考える

プロは車を個別の部品ではなく**系統（システム）**で考える。一つの系統内の部品は連動しているので、1つだけ替えても効果が限定的な場合がある。

```
┌─────────────────────────────────────────────────────┐
│                    系統別整備マップ                     │
├──────────┬──────────────────────────────────────────┤
│ 冷却系    │ ウォーターポンプ → サーモスタット →         │
│          │ エクスパンションタンク → ホース類 →          │
│          │ クーラント                                 │
├──────────┼──────────────────────────────────────────┤
│ 点火系    │ スパークプラグ → イグニッションコイル →      │
│          │ バルブカバーガスケット（オイル侵入防止）       │
├──────────┼──────────────────────────────────────────┤
│ 吸気系    │ エアフィルター → チャージパイプ →            │
│          │ インタークーラー → スロットルバルブ           │
├──────────┼──────────────────────────────────────────┤
│ オイル系   │ エンジンオイル → フィルター →               │
│          │ フィルターハウジングガスケット →              │
│          │ バルブカバーガスケット                       │
├──────────┼──────────────────────────────────────────┤
│ ブレーキ系 │ パッド → ローター → センサー →             │
│          │ フルード → ホース                          │
├──────────┼──────────────────────────────────────────┤
│ 足回り    │ アーム類（ブッシュ一体） → サブフレーム       │
│          │ ブッシュ → ハブベアリング → ダンパー →       │
│          │ マウント類 → アライメント                    │
├──────────┼──────────────────────────────────────────┤
│ 駆動系    │ M-DCTオイル → デフオイル →                 │
│          │ クラッチ → プロペラシャフト →                │
│          │ ドライブシャフトブーツ                       │
├──────────┼──────────────────────────────────────────┤
│ マウント系 │ エンジンマウント → ミッションマウント →      │
│          │ サブフレームブッシュ（足回りと重複）           │
└──────────┴──────────────────────────────────────────┘
```

**プロの視点：**
> 「ウォーターポンプだけ替えてサーモスタットを替えないお客さんがいる。
> 半年後にサーモが壊れてまた入庫。工賃が二重にかかる。
> 同じ系統は同時にやるのが鉄則。特に"外す手間が共通"の部品は絶対に一緒」


### 2-2. 「同時施工」で工賃を節約する

M3の整備で最もコストに影響するのは**工賃**。部品代より工賃のほうが高いケースが多い。

```
【同時施工の効果（例：冷却系）】

  個別にやった場合：
  ├─ ウォーターポンプ交換    工賃 2.5万円  部品 4万円
  ├─ サーモスタット交換      工賃 1.5万円  部品 1.5万円
  ├─ エクスパンションタンク  工賃 1万円    部品 1.2万円
  ├─ クーラント交換         工賃 0.5万円  部品 0.5万円
  └─ 合計                  工賃 5.5万円  部品 7.2万円  → 12.7万円

  同時施工した場合：
  ├─ 上記すべて同時         工賃 3万円    部品 7.2万円  → 10.2万円
  └─ 節約額: 約2.5万円
```

**同時施工が特に有効な組み合わせ：**

| メイン作業 | 一緒にやるべき作業 | 理由 |
|-----------|------------------|------|
| ウォーターポンプ交換 | サーモスタット、エクスパンションタンク、クーラント、**クランクハブ強化キット** | フロント周りを開ける作業が共通 |
| ブレーキパッド交換 | ローター、センサー、フルード | ホイールを外す作業が共通 |
| 足回りアーム一式交換 | サブフレームブッシュ、ハブベアリング、エンジン/ミッションマウント、アライメント | 車体をリフトアップして大規模に分解する |
| バルブカバーガスケット交換 | スパークプラグ、イグニッションコイル（必要時） | カバーを外す作業が共通 |
| M-DCTオイル交換 | デフオイル交換 | リフトアップが共通、作業時間も短い |


### 2-3. プロが「整備の順番」を決める基準

**優先度の判断マトリクス：**

```
          安全への影響
     低 ──────────── 高
  高│ ④予防整備     ②最優先     │壊
  　│（冷却系、     （ブレーキ、  │れ
  　│ クランクハブ） タイヤ）     │た
  　│                           │と
  　│                           │き
  　│ ③様子見       ①即対応     │の
  低│（コスメ系、   （警告灯、    │被
  　│ 快適装備）    異音・異臭）  │害
     ────────────────────────
```

- **①即対応**: 安全に直結＋被害拡大リスク → 最速で入庫
- **②最優先**: 安全に直結だが今すぐ壊れるわけではない → 次回入庫で確実に
- **④予防整備**: 壊れたとき被害大だが安全には直結しない → 計画的に実施
- **③様子見**: 安全・被害ともに低い → 余裕があるときに

**プロの視点：**
> 「お客さんは"全部一度にやりたい"か"必要最低限だけ"の二択になりがち。
> プロは"今やらないとまずいもの"と"次でいいもの"と"一緒にやると得するもの"を
> 組み合わせて最適な入庫計画を作る。1回の入庫で効率よく、でも無駄な作業はしない」

## 第4章：プロが見ている「数値」と「基準」

### 4-1. プロが測定する項目と判断基準

オーナーが「まだ大丈夫ですか？」と聞くと、プロは以下の数値で判断している：

**ブレーキ**
| 測定項目 | 新品時 | 交換推奨 | 危険域 |
|---------|--------|---------|--------|
| フロントパッド残量 | 約12mm | 3mm以下 | 1mm以下（警告灯点灯） |
| リアパッド残量 | 約10mm | 3mm以下 | 1mm以下 |
| ローター厚（F） | 34mm（新品） | 摩耗限度に近づいたら | 刻印の最小厚以下 |
| フルード沸点 | ドライ230°C以上 | ウェット170°C以下 | ウェット150°C以下 |

**エンジン**
| 測定項目 | 正常値 | 注意域 | 対応 |
|---------|--------|--------|------|
| オイル分析・銅（Cu） | 〜15ppm | 20ppm以上 | ベアリング摩耗の兆候。頻度を上げて監視 |
| オイル分析・鉛（Pb） | 〜10ppm | 15ppm以上 | 同上 |
| オイル消費量 | 1L/10,000km以内 | 1L/5,000km | ピストンリング等の点検 |
| クーラント量 | MAX-MIN間 | MINに近い | 漏れ点検 |
| ブースト圧 | 設定値通り | 設定値に達しない | チャージパイプ、ウェイストゲート点検 |

**足回り**
| 測定項目 | 正常 | 交換推奨 |
|---------|------|---------|
| ブッシュの状態 | ヒビなし、弾力あり | ヒビ・変形・硬化 |
| ハブベアリング | ガタなし、異音なし | ガタあり or ゴー音 |
| ダンパー | オイル漏れなし | 漏れ or 抜け |
| タイロッドエンド | ガタなし | ガタあり |

**プロの視点：**
> 「数値で判断するから"まだ大丈夫"か"もうダメ"かが明確になる。
> 感覚だけで判断すると、人間は"まだいけるだろう"方向にバイアスがかかる。
> 特にブレーキパッドとオイル分析は数値が全て」


### 4-2. プロが「音」と「匂い」で見つけるもの

数値化できない領域でもプロは情報を拾っている：

| 感覚 | 何を感じているか | 疑う箇所 |
|------|----------------|---------|
| 低速で「コトコト」 | サスペンション通過時の小さい音 | スタビリンク、アッパーマウント |
| 走行中「ゴー」 | 速度に比例する低い連続音 | ハブベアリング、タイヤの偏摩耗 |
| ブレーキ時「キー」 | 高音の金属音 | パッド残量限界、ローター段付き |
| 加速時「カリカリ」 | 高負荷時の異音 | ノッキング、ベアリング |
| シフト時「ガコン」 | 変速ショック | M-DCTオイル劣化、メカトロ |
| 甘い匂い | クーラントの匂い | クーラント漏れ |
| 焦げ臭い | 摩擦系の異常 | ブレーキ引きずり、クラッチ |
| 排気ガスの色 | 白煙/青煙 | 白＝クーラント混入、青＝オイル燃焼 |

## 第5章：油脂類（フルード）の整備 — 地味だが最重要

### なぜ油脂類が最重要なのか

車の中で**金属同士が接触する場所には必ず油脂類が入っている**。油脂類が劣化すると金属部品が摩耗し、高額部品の寿命を縮める。

```
【油脂類が守っているもの】

  エンジンオイル ──→ エンジン内部（ベアリング、ピストン、カム）
                     劣化 → ベアリング摩耗 → エンジンブロー（100万円〜）

  M-DCTオイル ──→ ギア、クラッチ、メカトロニックユニット
                   劣化 → シフト悪化 → メカトロ故障（100万円〜）

  デフオイル ──→ デフギア、アクティブMデフのクラッチ
                  劣化 → 異音 → ギア損傷（30〜50万円）

  ブレーキフルード ──→ ブレーキシステムの油圧伝達
                       劣化 → 沸点低下 → ベーパーロック（ブレーキ効かない）

  クーラント ──→ エンジン冷却
                  劣化 → 防錆性能低下 → ラジエーター/ポンプの腐食

  パワステフルード ──→ ステアリングの油圧補助（F80は電動のため該当なし）
```

**「安い保険」としての油脂類交換：**
```
  オイル交換1回 = 1〜2万円
  エンジンブロー = 100万円〜

  M-DCTオイル交換1回 = 1〜2万円
  メカトロ故障 = 100万円〜

  デフオイル交換1回 = 1〜2万円
  デフ修理 = 30〜50万円

  → 油脂類の定期交換は、最もコスパの高い予防整備
```

### F80 M3の油脂類交換スケジュール

| 油脂類 | BMW推奨 | プロの推奨（M3/高走行） | 理由 |
|--------|--------|----------------------|------|
| エンジンオイル | 15,000km or 2年 | **7,500〜10,000km or 1年** | S55はターボで熱負荷が高い。BMW推奨はロングライフ前提で長すぎる |
| オイルフィルター | オイル交換毎 | オイル交換毎 | — |
| M-DCTオイル | 「無交換」（公式） | **40,000〜60,000km** | BMWの「無交換」は寿命までの意味。延命するなら交換必須 |
| デフオイル | 「無交換」（公式） | **40,000〜60,000km** | 同上。アクティブMデフはクラッチの摩耗粉が出る |
| ブレーキフルード | 2年 | **2年** | これはBMW推奨通りでOK |
| クーラント | 4年 or 80,000km | **4年 or 60,000km** | 経年で防錆添加剤が劣化 |

**プロの視点：**
> 「BMWの"無交換"は信じてはいけない。あれは"保証期間内は持つ"という意味。
> M-DCTオイルもデフオイルも、交換すると明らかにフィーリングが変わる。
> 逆に言えば、それだけ劣化しているということ。
> うちに来るM3で10万km超えてM-DCTオイル未交換の個体は、
> 抜いたオイルが真っ黒で金属粉がキラキラしている」

> 「エンジンオイルもBMWの15,000kmは長すぎる。ターボエンジンは熱負荷が桁違い。
> 7,500kmで替えてるオーナーのS55と、15,000kmまで引っ張ってるS55は、
> 10万kmの時点でエンジン内部のコンディションが全然違う」

## 第6章：足回り整備の特殊性

### 6-1. なぜM3の足回りは「一式交換」なのか

前章の vehicle-dynamics.md で解説した通り、M3の足回りはすべてが連携して機能する精密なシステム。

```
【一部だけ交換した場合の問題】

  新品アーム（硬い）
     ↕  ← 剛性の差
  古いアーム（柔らかい）

  → 左右で挙動が異なる
  → アライメントが左右で合わない
  → 最悪、古いアーム側だけ偏摩耗が進行
```

**プロの視点（Sun Beam）：**
> 「F80のブッシュはアームに圧入されてるから、ブッシュだけ替えるってことができない。
> アームごと交換になる。で、片側だけ替えると左右のバランスが崩れる。
> 11万km走った車は全部が均等に劣化して、ある意味バランスが取れている。
> 中途半端にやるくらいなら一気にやったほうが結果はいい」

### 6-2. 足回り交換の作業フロー

プロの足回りリフレッシュは以下の手順で行われる：

```
  1. 現状の確認・記録
     └─ 各部のガタ、ブッシュの状態、異音を記録

  2. 部品の手配
     └─ 全アーム、ブッシュ、ベアリング、ボルト/ナット類
     └─ 納期確認（BMW純正は在庫あれば数日、海外取り寄せは2〜4週間）

  3. 分解
     └─ ホイール → ブレーキキャリパー退避 → ドライブシャフト切り離し
     └─ アーム類取り外し → サブフレーム降ろし（必要に応じて）

  4. 交換
     └─ サブフレームブッシュ交換 → サブフレーム搭載
     └─ 新品アーム取り付け → ハブベアリング交換
     └─ ドライブシャフト接続 → ブレーキ復元

  5. アライメント調整
     └─ 4輪アライメント測定・調整
     └─ M3の推奨値にセット

  6. 試走・最終確認
     └─ 各速度域での直進性、異音の有無を確認
     └─ ボルトの増し締め（走行後の初期なじみ後）

  作業日数：通常2〜4日（サブフレーム脱着を含む場合）
```

### 6-3. 足回り交換後に何が変わるか

オーナーが体感できる変化：

| 項目 | Before（11万km） | After（新品交換後） |
|------|-----------------|-------------------|
| ステアリングのセンター | わずかに曖昧 | ピタッと決まる |
| 直進安定性 | ふらつく感じ | 矢のように直進 |
| コーナリング | 曖昧、ワンテンポ遅れる | 切った瞬間にノーズが入る |
| 路面のフィードバック | ぼやけている | 路面の状況が手に伝わる |
| 乗り心地 | ゴツゴツ or 落ち着かない | 硬いが角が丸い（しなやか） |
| 異音 | コトコト、ギシギシ | 無音 |

**プロの視点：**
> 「足回り一式やったお客さんは全員"別の車になった"と言う。
> 11万kmのブッシュは見た目でわかるくらいヒビが入っている。
> 新車のときのハンドリングを知らない中古オーナーが多いから、
> 交換後に初めて"M3ってこんな車だったのか"と気づく。
> これが一番費用対効果が高い整備だと個人的には思う」

## 第7章：「ショップ選び」と「コミュニケーション」

### 7-1. M3に適したショップの条件

| 条件 | 理由 |
|------|------|
| BMW Mモデルの整備実績が豊富 | 専用工具・診断機・ノウハウが必要 |
| ISTA（BMW純正診断機）を持っている | コーディング、アダプテーション、エラー履歴の読み出しに必須 |
| 作業の説明を嫌がらない | 何をなぜやるか説明してくれるショップは信頼できる |
| 同時施工の提案をしてくれる | オーナーのコスト最適化を考えてくれている証拠 |
| 「やらなくていい」も言ってくれる | 不要な作業を勧めないのはプロの誠実さ |

### 7-2. ショップに行くときの伝え方

**プロが助かる伝え方：**
```
  ✗ 「なんか調子悪い気がする」
  ○ 「60km/h以上で右にステアリングを切ると、左前から"コトコト"音がする。
     雨の日は出ない。3ヶ月前くらいから気づいた」
```

**伝えるべき情報：**
1. **いつから** — 最近か、以前からか
2. **どんなとき** — 速度域、操作（ブレーキ/加速/ステアリング）、路面状態
3. **どんな症状** — 音（擬音語で具体的に）、振動、匂い、警告灯
4. **再現性** — 毎回か、たまにか
5. **変化** — 悪化しているか、変わらないか

**プロの視点：**
> 「"なんか変"だけだと、こっちが全部チェックしないといけないから時間がかかる。
> でも"右コーナーで、60km/h以上で、コトコト"と言われたら、
> まずスタビリンクとアッパーマウントを疑えるから診断が速い。
> お客さんが普段から車の変化に気づいてメモしてくれると本当に助かる」

> この順番で計画的に進めるのが賢い」
